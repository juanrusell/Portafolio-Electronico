¿Qué es React?

React es una biblioteca de JavaScript desarrollada por Facebook para construir interfaces de usuario. Se basa en una arquitectura orientada a componentes, permitiendo que cada parte de la interfaz (botones, formularios, listas, etc.) sea modular y reutilizable.

Características principales:

- Componentes: Facilitan el mantenimiento y reutilización de partes de la interfaz.

- Virtual DOM: Mejora el rendimiento al actualizar solo los elementos que cambian.

- Unidireccionalidad de datos: Garantiza un flujo claro y predecible de información.

- JSX: Permite escribir HTML dentro de archivos JavaScript de manera intuitiva.

- React Hooks: Introducidos en React 16.8, permiten manejar estado y lógica en componentes funcionales.
