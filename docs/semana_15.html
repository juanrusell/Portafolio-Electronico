Temas y actividades

Temas aprendidos:

Esta semana se desarrolló un sistema inteligente básico utilizando Python, con enfoque en la inteligencia artificial aplicada a la toma de decisiones y análisis de datos.

¿Qué es un sistema inteligente?

Es un software capaz de razonar, aprender o actuar de manera autónoma en función de datos, reglas o comportamientos. Utiliza conceptos de IA como machine learning, lógica difusa, sistemas expertos, redes neuronales, entre otros.

Enfoque del proyecto:

Se desarrolló un sistema experto simple para asesoría médica básica (triage), capaz de dar recomendaciones según síntomas ingresados por el usuario.

Flujo básico del sistema:

1. El usuario ingresa sus síntomas (ej. fiebre, tos, dolor de garganta).

2. El sistema analiza los síntomas con reglas predefinidas.

3. Retorna una sugerencia (por ejemplo, "Podrías tener una gripe leve. Descansa e hidrátate.").

Fragmento de código:


def diagnosticar(sintomas):
    if "fiebre" in sintomas and "tos" in sintomas:
        return "Posible gripe. Se recomienda reposo y abundante líquido."
    elif "dolor de cabeza" in sintomas and "visión borrosa" in sintomas:
        return "Consulta médica urgente recomendada."
    else:
        return "Síntomas no concluyentes. Visita un especialista."

Bibliotecas utilizadas:

— tkinter: para crear una interfaz gráfica básica.

— pandas y numpy: en caso se analicen datasets de salud.

— sklearn: si se desea aplicar un modelo de aprendizaje automático.

Aplicaciones reales:

  • Chatbots inteligentes para atención al cliente.
  • Sistemas de diagnóstico médico preliminar.
  • Recomendadores de productos o servicios.
  • Asistentes personales y de productividad.

Reflexión: Aprendí cómo Python facilita la creación de sistemas inteligentes a pequeña escala. El uso de reglas y lógica para generar recomendaciones automáticas representa la base de la inteligencia artificial práctica que vemos a diario.
