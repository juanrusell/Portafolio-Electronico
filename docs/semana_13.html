Temas y actividades

Temas aprendidos:

Se abordó el desarrollo backend utilizando Laravel, uno de los frameworks más populares de PHP, conocido por su sintaxis elegante y herramientas integradas para crear aplicaciones web robustas de forma eficiente.

¿Qué es Laravel?

Laravel es un framework de código abierto basado en PHP que facilita la creación de aplicaciones web siguiendo el patrón MVC (Modelo–Vista–Controlador). Proporciona herramientas modernas como migraciones, rutas, controladores, plantillas Blade y Eloquent ORM para trabajar con bases de datos.

Características principales:

— Enrutamiento simple y flexible con archivos de rutas como web.php y api.php.

— Controladores que separan la lógica del negocio de las vistas.

— Plantillas Blade que permiten mezclar HTML y lógica PHP fácilmente.

— Eloquent ORM que simplifica las consultas a la base de datos con clases modelo.

— Mecanismos de autenticación y autorización listos para usar.

Flujo de desarrollo básico:

1. Crear un nuevo proyecto con laravel new proyecto o composer create-project.

2. Definir rutas en routes/web.php.

3. Crear controladores con php artisan make:controller.

4. Crear modelos y migraciones para manejar la base de datos.

5. Construir vistas con Blade en la carpeta resources/views.

6. Ejecutar migraciones y probar con servidor local usando php artisan serve.

Herramientas adicionales:

— Laravel Tinker para pruebas rápidas en la consola.

— Laravel Artisan, CLI propia para generar código y ejecutar tareas.

— Laravel Breeze y Laravel Jetstream para implementar autenticación moderna.

Reflexión: Laravel simplifica el desarrollo backend en PHP al proporcionar una estructura clara y herramientas integradas que agilizan tareas comunes como el enrutamiento, el acceso a bases de datos y la autenticación. Este framework es ideal para construir proyectos web modernos, escalables y seguros.
