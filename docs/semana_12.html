Temas y actividades

Temas aprendidos:

Exploración de una solución tecnológica real que aplica inteligencia artificial y visión por computadora para resolver un problema cotidiano: el desperdicio de alimentos.

¿Qué es FridgeSnap?

FridgeSnap es una aplicación móvil desarrollada en Java y XML para Android. Permite al usuario tomar una foto de un alimento, identificarlo mediante un modelo de deep learning y sugerir recetas basadas en los ingredientes disponibles. La app se conecta a la API de Spoonacular para obtener recetas en tiempo real.

Funcionamiento general:

1. El usuario inicia sesión.

2. Toma una fotografía de un alimento (como una manzana o zanahoria).

3. El modelo DenseNet identifica el alimento y lo agrega a una lista de ingredientes.

4. El usuario puede agregar más alimentos, ya sea con fotos o manualmente.

5. La app sugiere recetas basadas en los ingredientes registrados.

Tecnología utilizada:

— Modelo de clasificación DenseNet entrenado con 40,000 imágenes de 20 clases de alimentos.

— Backend en Python para procesamiento de imágenes.

— App móvil construida en Java/XML.

— API externa Spoonacular para las recetas.

Impacto del proyecto:

— Ayuda a reducir el desperdicio de alimentos en los hogares.

— Promueve hábitos sostenibles y responsables.

— Aporta un nuevo dataset útil para investigación académica.

— Enseña planificación de comidas y cocina saludable.

Limitaciones:

Actualmente, la app no puede estimar la cantidad de alimento en la imagen, lo cual puede afectar la precisión de las recetas sugeridas.

Reflexión: Este análisis permitió comprender cómo una aplicación móvil puede integrar IA, visión por computadora y APIs externas para resolver problemas sociales. FridgeSnap representa un excelente ejemplo de cómo la tecnología puede tener un impacto ambiental, económico y educativo positivo.
