Temas aprendidos: HTML estructura el contenido web mediante etiquetas como h1, p, etc.

Historia de HTML:
HTML (HyperText Markup Language) fue introducido por Tim Berners‑Lee en 1991. La evolución clave:
HTML 1.0 (1991) → HTML 2.0 (1995) → HTML 3.2 (1997) → HTML4 (1997) → XHTML (2000) → HTML5 (2014‑presente). Cada versión añadió soporte para más diseño, estilo, multimedia y accesibilidad.

Etiquetas y atributos:

Los atributos proporcionan información adicional sobre un elemento. Ejemplos:
ID (único por elemento), class (puede repetirse), src (ruta de una imagen), alt (texto alternativo), width/height (dimensiones)…
Las etiquetas permiten definir encabezados, párrafos, imágenes, enlaces, listas, formularios y más.

Ejercicios de laboratorio: Construcción de una página básica con etiquetas semánticas, listas, imágenes y vínculos.
