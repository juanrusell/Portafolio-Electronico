Temas y actividades

Temas aprendidos:

Esta semana exploramos el framework Flask, una herramienta ligera pero poderosa para desarrollar aplicaciones web con Python, ideal para APIs RESTful y proyectos de rápida implementación.

¿Qué es Flask?

Flask es un microframework de Python que permite construir aplicaciones web de forma rápida y sencilla. A diferencia de frameworks más grandes, como Django, Flask no impone una estructura rígida, lo que brinda flexibilidad al desarrollador.

Características principales:

— Ligero y minimalista: solo lo esencial para empezar, pero extensible mediante extensiones.

— Soporte para rutas, peticiones y respuestas HTTP de forma sencilla.

— Integración fácil con bases de datos como SQLite, MySQL o PostgreSQL.

— Ideal para APIs RESTful y microservicios.

Flujo de desarrollo básico:

1. Crear un entorno virtual y activar.

2. Instalar Flask con pip install flask.

3. Crear un archivo app.py con el servidor básico y rutas.

4. Ejecutar con flask run y acceder desde el navegador.

5. Agregar lógica, formularios, bases de datos y renderización de plantillas.

Ejemplo de código básico:


from flask import Flask

app = Flask(__name__)

@app.route('/')
def inicio():
    return "¡Hola desde Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Extensiones útiles:

— Flask-SQLAlchemy: para trabajar con bases de datos relacionales.

— Flask-WTF: para manejar formularios y validaciones.

— Flask-Login: para autenticación de usuarios.

Reflexión: Flask permite construir aplicaciones web modernas con una curva de aprendizaje accesible. Su simplicidad y flexibilidad lo convierten en una excelente opción para proyectos pequeños, APIs y prototipos, así como para desarrolladores que buscan tener mayor control del backend.
