Resumen de aprendizajes

Temas aprendidos: Emmet es una herramienta para escribir código HTML y CSS rápidamente usando abreviaciones.

Emmet es un plugin que existe para la mayoría de los editores de texto comunes, pensado para mejorar el flujo de trabajo. Este plugin está disponible de forma predeterminada en VS Code y facilita la generación de código HTML y la apertura de etiquetas.

Ejercicios de laboratorio: Combinaciones como div>header+main+footer, listas con clases, etc.

Ejemplo práctico:
Escribir el generador Emmet para una estructura HTML semántica con IDs del mismo nombre y contenido de texto, agrupados en un contenedor:

El resultado del generador Emmet fue:

div.container>header#header{header}+nav#nav{nav}+section#section{section>article#article{article}^}+aside#aside{aside}+footer#footer{footer}

Reflexión: Aprendí cómo mejorar mi velocidad al crear estructuras HTML usando Emmet. Las abreviaciones me permiten generar mucho código en menos tiempo, lo cual es muy útil para proyectos web.
